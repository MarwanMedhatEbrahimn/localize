How to Implement Angular Localization

We need to add i8n attribute to the tag that have the sentences we need to translate

We recommend to use id by adding a value to the attribute i18n "@@ID" (the ID doesn't have to be the same as the translated sentence)

Sometimes we need to translate some values of attributes like the title and the alt of an image's tag or the placeholder of an input field

Sometimes we need translate part of sentences in tag, it this time we can use ng-container tag

Run " ng extract-i18n --format=json " in terminal To create source language file in the root directory of your project

Create a locale directory and set it in src directory

Copy the source language file and paste it at locale directory

Rename it to messages.languageName.json for example messages.ar.json

In this repo i rename it to main.ar.json it not problem to change it as you went

In main.ar.json you will find the IDs you added before in attribute i18n you need after that change value of this IDs to translated of sentences

After that write in angular.json this code in projects object

    //In Object that have the same name of project
    "localize": {
      .
      .
      "i18n": {
      //Here we write the name of the original language of web site
        "sourceLocale":"en-US",
        "locales": {
          //Here we set the name of language
          "ar":{
            //Here we write the path of the translated file of this language
            "translation": "src/locale/main.ar.json",
            //Here we set the domain name to access the translated server
            "baseHref": "ar/"
          }
          //Here we can set more languages 
        }
      },
      .
      .
    }

In the last step you need to build project use " ng build --localize "

this cmd will give you in dist directory, directory for every language you add in angular.json
